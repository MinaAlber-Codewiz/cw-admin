Select Input Fields Select Input Fields has different styles to choose from, check the below examples. Select Input Fields must be wrapped inside a div with the class "form-field" followed by the helper class for the style of the input textarea field needed.
Block Level Style Use the following class "style-block" to create Block Level Style Select Input Fields.
Block Level Style with Icon Use the following class "style-block" followed by the following classes "icon" and "icon-puzzle" for Icons.
Block Level Border Bottom Style Use the following class "style-block-border-bottom" to create Block Level Border Bottom Style Select Input Fields.
Block Level Border Bottom Style with Icon Use the following class "style-block-border-bottom" followed by the following classes "icon" and "icon-puzzle" for Icons.
Inline Block Level Style Use the following class "style-inline-block" to create Inline Block Level Select Input Fields.
Inline Block Level Style with Icon Use the following class "style-inline-block" followed by the following classes "icon" and "icon-puzzle" to create Inline Block Level Select Input Fields with Icons.
Floating Label Style Use the following class "style-float-label" to create Floating Label Select Input Fields.
Floating Label Style with Icon Use the following class "style-float-label" followed by the following classes "icon" and "icon-puzzle" for Icons.
Floating Label Legend Style Use the following class "style-float-label" in addition to class "float-label-legend" to create Floating Label Legend Select Input Fields.
Floating Label Legend Style with Icon Use the following class "style-float-label" in addition to class "float-label-legend" followed by the following classes "icon" and "icon-puzzle" for Icons.
Floating Label Border Bottom Style Use the following class "float-label-border-bottom" in addition to class "style-float-label" to create Floating Label Border Bottom Select Input Fields.
Floating Label Border Bottom Style with Icon Use the following class "float-label-border-bottom" in addition to class "style-float-label" followed by the following classes "icon" and "icon-puzzle" for Icons.
Select Input Passed Validation Use the following class "field-pass" with the form-field div to change label and input fields to color green. In addition to adding a div with a class "input-success-msg" to show the Success Message if needed.
