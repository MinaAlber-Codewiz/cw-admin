Checkbox Input Fields Checkbox Input Fields has different styles to choose from, check the below examples. A Checkbox Input Field along with its label must be wrapped in a div with the class "checkbox-input". The Div containing the checkbox input field and its label must be wrapped inside a div with the class "form-field" followed by the helper class for the style of the input Checkbox field needed.
