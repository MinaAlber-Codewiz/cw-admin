Accordions Accordions must be wrapped in a master div with the class "accordion" which acts as a container, so no need for the container class. Each Accordion Section should have a title and a body. The Accordion Section title should be a div with the class "accordion-title" containing its title. The Accordion Section Body should be wrapped in a div with the class "accordion-body". For adding icons to the Accordion Section Title, add the following classes "icon" and "icon-puzzle" in addition to the title div "accordion-title".
