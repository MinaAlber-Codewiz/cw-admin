Modals For Modals, the element that is going to open the modal needs to be created first with the Id Target referencing to the modal to open. The Modal needs to have its Targeted Id for opening it when the element that is going to open the modal is clicked. The Modal must have a reference id with the following attribute data-remodal-id="ID Name" which will be referenced in the element that is going to open the Modal. The Element that is going to open the modal must have the attribute with the modal id that it is going to open like the following data-remodal-target="Modal ID Name". The Close button for the Modal or any other element that is going to trigger a close event must have the following attribute and value data-remodal-action="close". There are Multiple Options for the Modal, which is passed to the main Modal div in the attribute data-remodal-options, each option is separated with a space and takes a boolean value of either false or true. hashTracking Option: Enables/Disables the hash in the address bar for the modal. closeOnOutsideClick Option: Enables/Disables closing the modal if clicked outside of the modal frame. closeOnEscape Option: Enables/Disables closing the modal if the ESC key is pressed. The Modal body div with the class "modal-body" acts the same as a container, so a div for containing the rows and columns is not needed.
